## 🚩 Build & Run
1. 빌드
`[Build Setting] → add GotOfArcher.scene → Build`

2. 빌드된 파일 실행  `God_Of_Archer.exe`
<br>

## 🎮 Control & Play
### | Control
|조작|버튼|
|:---:|:---:|
|움직임|W,A,S,D|
|달리기|Shift|
|활 줍기|E|
|활 쏘기|마우스 좌클릭|
|활 초기화|R|

### | Play Method
```
1. 마스터 클라이언트가 서버를 연다.
2. 나머지 플레이어가 참가한다.
3. 4인이 모이면 게임이 시작된다.
```
